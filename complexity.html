Complexity class or designer drug?

Click each link to see if you're correct.

  • MBT
    Drug: N-butyl-N-methyltryptamine is a tryptamine psychedelic.
  • CZK
    Complexity class: Computational Zero-Knowledge is "same as SZK, except that now the two distributions are merely required to be computationally indistinguishable by any BPP algorithm; they don't have to be statistically close."
  • α-MT
    Drug: α-methyltryptamine is a tryptamine psychedelic. Rumor has it that Neal Cassady called this "the Rolls Royce of psychedelics".
  • β-D
    Drug: 3,4,5-trimethoxy-β,β-dideuterophenethylamine is a derivative of mescaline where two "light" hydrogens are replaced with deuterium.
  • LWPP
    Complexity class: Length-Dependent Wide PP is "the class of decision problems solvable by an NP machine such that
    1. If the answer is 'no,' then the number of accepting computation paths exactly equals the number of rejecting paths.
    2. If the answer is 'yes,' then the difference of these numbers equals a function f(|x|) computable in polynomial time (i.e. FP). Here |x| is the length of the input x, and 'polynomial time' means polynomial in |x|, the length of x, rather than the length of |x|."
  • TFMPP
    Drug: 3-trifluoromethylphenylpiperazine is a synthetic piperazine entactogen/empathogen, often packaged with BZP as an Ecstasy simulant.
  • QMIP
    Complexity class: Quantum Multi-Prover Interactive Proofs is "the quantum generalization of MIP, and the multi-prover generalization of QIP."
  • BZ
    Drug: 3-quinuclidinyl benzilate is an anticholinergic deliriant hallucinogen. The US Army tested BZ on humans at Edgewood Arsenal for use as a possible chemical warfare agent. It inspired the 1990 psychological horror film Jacob's Ladder.
  • TFNP
    Complexity class: is "the class of function problems of the following form: Given an input x and a polynomial-time predicate F(x,y), output any y satisfying F(x,y). (Such a y is promised to exist.)"
  • BPL
    Complexity class: Bounded-Error Probabilistic L "has the same relation to L as BPP does to P".
  • mCPP
    Drug: 1-(m-chlorophenyl)piperazine is a 5-HT receptor agonist which has been known to induce migranes. It has also been found in pills sold as Ecstasy.
  • φ2P
    Complexity class: Second Level of the Symmetric Hierarchy, Alternative Definition is "the class of problems for which there exists a polynomial-time predicate P(x,y,z) such that for all x, if the answer on input x is 'yes,' then
    1. For all y, there exists a z for which P(x,y,z).
    2. For all z, there exists a y for which P(x,y,z)."
